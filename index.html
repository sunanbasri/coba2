Integrasi Numerik¶

Integral secara umum adalah bentuk operasi matematika yang merupakan kebalikan dari operasi turunan fungsi. Selain itu, integral juga dapat diartikan sebagai batas jumlah atau luas suatu luasan tertentu. Secara simbolis, integral dilambangkan dengan ʃ dx. Menurut pengertian tersebut, integral dibagi menjadi dua jenis, yaitu integral tertentu dan integral tak tentu. [^1]

Contoh Penerapan integral dalam sehari-hari, antara lain menentukan luas dari suatu bidang, menentukan volume benda-benda putar dan lain sebagainya.[^2]

Integrasi numerik atau bisa juga disebut dengan perhitungan integral merupakan dasar perhitungan yang banyak digunakan dalam banyak hal seperti digunakan pada kalkulus. Tujuan dari integrasi numerik itu sendiri yaitu untuk menyelesaikan integral yang sulit diselesaikan secara analitik.[^3]

Integral tak tentu merupakan kebalikan dari turunan atau lebih umum disebut anti turunan. Hasil integral tak tentu dari fungsi tersebut adalah suatu fungsi baru yang belum mempunyai nilai tentu atau pasti, karena masih terdapat variabel dalam fungsi baru tersebut Integral tentu adalah hasil dari penjumlahan luas daerah yang dilingkupi oleh kurva suatu persamaan tertentu. perbedaan dengan integral tak tentu, integral tentu sudah memiliki nilai tetap, dikarenakan batas yang ditentukan jelas.[^4]

Aturan Trapezoid (Trapezoida Rule)¶

Aturan Trepezoid merupakan metode untuk mendekati integrasi numerik dengan polinomial orde pertama. Dalam metode ini, kurva melengkung mendekati garis lurus, membuat bentuk di bawah kurva seperti trapesium

Algoritma dengan metode trapezoid¶
  1. Pertama Definisikan nilai y = f(x)
  2. Menentukan batas bawahnya (x0) dan batas atasnya (xn).
  3. Tentukan jumlah pembagi (n)
  4. Hitung h = (xn - x0) / n
  5. Hitung integrasi = f(x0) + f(xn)
  6. for i=1 : n
  7. Hitung k = x0 + i*h
  8. Hitung integrasi = integrasi + 2 * f(k)
Code program Python dengan Metode Trapezoid¶
def f(x):
    #nilai f(x)
    return 1/(1 + x**2)

def trapezoid(x0,xn,n):
    h = (xn - x0) / n
    integrasi = f(x0) + f(xn)

    for i in range(1,n):
        k = x0 + i*h
        integrasi = integrasi + 2 * f(k)

    integrasi = integrasi * h/2
    return integrasi

#input user    
btsbawah= float(input("isi Batas Bawah: "))
btsatas = float(input("isi Batas Atas: "))
jmlinter = int(input("isi Banyaknya Interval: "))

#menjalankan program
hsl = trapezoid(btsbawah, btsatas, jmlinter)
print("Hasil Integrasi Metode Trapezoid: %0.6f" % (hsl) )
